DYNAMIC SPORTS STATISTICS AND PREDICTIONS

A Dynamic Sports Prediction and Statistics Viewing Dashboard made in Google Sheets.

OVERVIEW

I developed a dynamic sports statistics dashboard using Google Sheets, designed to track and analyze various statistics during the season for a sports league. This dashboard offers real-time data updates, a custom power ranking system, and score predictions for selected teams. It combines data from various sources, Solver optimization, and Monte Carlo simulations to provide a comprehensive analysis of the league's performance.

TECHNOLOGIES USED

This project made uses of the following tools:

  • Google Sheets
  • Solver
  • Monte Carlo
  • Data Integration
  • Advanced Formulas
  • Bradley Terry
KEY FEATURES
  • Real-time Data Integration: The dashboard is connected to external data sources, ensuring that it is always up-to-date with the latest statistics, scores, and performance metrics.

  • Custom Power Raking System: I created a unique power ranking system within the dashboard. This system utilizes Solver, a powerful optimization tool, to calculate the relative strengths of teams based on historical performance, enabling users to see how teams compare in real-time.

  • Monte Carlo Simulations: To enhance the predictive capabilities of the dashboard, I incorporated Monte Carlo simulations. This statistical technique allows users to simulate various game scenarios and assess the likelihood of different outcomes, providing valuable insights into potential future performance.

  • User-Selectable Teams:: Users can interact with the dashboard by selecting specific teams of interest. The dashboard provides in-depth statistics, analysis, and predictions for the chosen teams, making it a versatile tool for fans, analysts, and team management.

  • Score Predictions: A backend prediction model is integrated into the dashboard, enabling users to input specific game scenarios and receive score predictions. This feature adds a layer of sophistication to the dashboard, making it an essential tool for anyone interested in predicting game outcomes.
